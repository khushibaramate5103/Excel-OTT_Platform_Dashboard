# 📺 OTT Platform Dashboard – Excel Project

# 📌 Description

An interactive OTT analytics dashboard created in Microsoft Excel to analyze streaming platform performance. The dashboard visualizes user activity, watch time, ratings, device usage, and subscription patterns across platforms such as Netflix, Amazon Prime Video, Disney+, HBO Max, Hulu, and Apple TV+.
It uses pivot tables, charts, and slicers to provide an interactive way to explore OTT data insights. 📊

# 📂 Dataset Information

🔹 User_ID – Unique identifier for each user

🔹 Platform – OTT service where content is streamed

🔹 Genre – Content category (Action, Drama, Comedy, etc.)

🔹 Show_Name – Title of the show or movie

🔹 Country – Country where the viewer is located

🔹 Rating – User rating for the content (1–5)

🔹 Watch_Time_Hours – Total hours watched by the user

🔹 Device – Device used for streaming (Mobile, Smart TV, Laptop/Desktop, Tablet)

🔹 Subscription_Type – Subscription plan used (Free Trial, Monthly, Yearly)

🔹 Date – Date of streaming activity

# 📊 Dashboard Visualizations

📈 Watch Time by Platform – Compares engagement across OTT services

🎭 Genre Analysis – Shows viewer interest across genres

🌍 Ratings by Country – Compares content ratings across regions

📱 Device Usage – Distribution of streaming devices

💳 Subscription Breakdown – Analysis of subscription types

📉 Platform Trends – Streaming trends over time

The purpose of this project is to analyze OTT platform data and present insights through an interactive Excel dashboard, demonstrating practical skills in data analysis and visualization.
