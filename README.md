# OTT Platform Dashboard (Excel)
# Project Overview

The OTT Platform Dashboard is an interactive data visualization project built using Microsoft Excel / WPS Office. The dashboard analyzes streaming platform performance and user behavior across multiple OTT services such as Netflix, Amazon Prime Video, Disney+, HBO Max, Hulu, and Apple TV+.

The goal of this dashboard is to transform raw OTT platform data into clear visual insights that help analyze watch time, subscription patterns, content ratings, and device usage. The dashboard enables users to filter and explore the data using interactive controls such as slicers.

# Dashboard Features
# 1. Total Watch Time by Platform

This bar chart displays the total number of watch-time hours for each OTT platform.
It allows comparison of platform engagement and helps identify which streaming service has the highest viewer activity.
